# 🦁 动物同花顺 - 森林股市模拟游戏

## 项目简介

动物同花顺是一个基于React的森林主题股票交易模拟游戏。玩家可以扮演不同的动物角色，在虚拟的股票市场中进行买卖交易，体验真实的股票交易流程。游戏包含丰富的AI动物玩家、季节天气系统、年度大会等特色功能。

---

## 🎮 游戏核心功能

### 角色系统
- **玩家角色选择**: 可选择7种不同的动物角色开始游戏
- **AI动物玩家**: 6个AI角色自动参与交易，每个角色有独特的交易风格
- **情绪系统**: 玩家盈亏会影响角色的情绪表现和对话

### 交易系统
- **委托交易**: 设置价格和数量进行挂单委托
- **订单撮合**: 买卖订单自动撮合成交（集合竞价机制）
- **冻结机制**: 委托时资金/股份会被冻结，撮合成功后转移
- **委托限制**: 每个玩家最多同时挂3个买入委托和3个卖出委托
- **快捷交易**: 支持涨停买、跌停卖、市价买、市价卖快捷按钮

### 价格系统
- **实时价格波动**: 股价会根据市场供需关系实时波动
- **月度涨跌停限制**: 委托价格不能超过月初价格的±20%（在下单时强制限制）
- **集合竞价成交**: 成交价取买卖双方的中间价，避免价格单向下跌
- **卖单参考买一价**: 动物提交卖单时会参考当前最高买价，促进成交

### 时间系统
- **游戏时间**: 游戏内时间从2000年3月1日开始
- **速度模式**: 支持1x（正常）、2x（4倍速）、4x（4倍速）三档速度
- **暂停功能**: 可随时暂停/继续游戏

### 季节与天气系统
- **四季轮换**: 春、夏、秋、冬四季自动轮换
- **天气变化**: 晴天、多云、雨天、雪天、大风、雾天等天气随机变化
- **播报系统**: 季节/天气变化时会有莫莫鸟的森林播报

### 年度大会系统
- **自动触发**: 每年1月1日自动触发年度大会
- **大会环节**: 开幕致辞、市场回顾、财富排行、慈善捐款、代表发言、精彩节目、闭幕
- **历史查看**: 可随时查看历年年度大会记录
- **慈善捐款**: 盈利动物向幼鸟保护基金会捐款，获得公告头条特权

### 公告与聊天系统
- **森林播报**: 莫莫鸟主播的实时森林播报
- **聊天室**: 展示动物们的交易自言自语、天气变化、季节变化等消息
- **大单播报**: 大额委托订单会有专门的播报

## 🎯 游戏规则详解

### 游戏目标
通过低买高卖股票，获得最大利润，成为森林首富！

### 盈亏计算
```
总市值 = 当前资金 + 冻结资金 + (持股 + 冻结股份) × 当前价格
盈亏 = 当前总市值 - 起始总市值
盈亏比例 = 盈亏 / 起始总市值
```

### 委托机制
1. **买入委托**: 
   - 输入委托价格和数量
   - 资金立即冻结（委托价 × 数量）
   - 等待撮合或撤销

2. **卖出委托**:
   - 输入委托价格和数量
   - 股份立即冻结
   - 等待撮合或撤销

3. **订单撮合**:
   - 每天自动撮合一次
   - 买入价 ≥ 卖出价时成交
   - 成交价为双方委托价的中间值

### 撤销机制
- 玩家可主动撤销委托
- 连续5轮未成交的委托会自动撤销
- 撤销后冻结资源返还

### 游戏失败
- 资金低于¥100时游戏失败
- 可重新开始游戏

## 🌟 特色功能详解

### 1. 森林播报系统
莫莫鸟作为森林主播，会实时播报：
- 季节变化（"啾啾~ 春天来了！"）
- 天气变化（"啾啾~ 下雨了，注意防滑！"）
- 大单委托（"大单预警！庄家虎买入1000股！"）
- 市场动态（"市场火热，交易活跃！"）

### 2. 聊天室系统
动物们会在聊天室发表各种消息：
- **交易自言自语**: 下单时的心理活动
- **天气变化反应**: 对天气变化的感慨
- **季节变化反应**: 对季节变化的感受
- **打工消息**: 出门打工和归来的消息
- **年度大会**: 大会期间的讨论

### 3. 打工系统
- **触发条件**: 动物有一定概率出门打工
- **打工概率**: 根据季节和资金状况调整
- **打工时长**: 1-3个月
- **打工收益**: 固定薪资 ¥100 - ¥20,000
- **资金影响**: 资金低于初始30%时打工概率翻倍

### 4. 市场流动性维护
当市场流动性不足时：
- 卖单不足时，AI会提高买价促进成交
- 买单不足时，AI会抛售股份提供流动性

### 5. 年度大会系统

#### 触发条件
- 每年1月1日自动触发
- 游戏时间满一年后触发

#### 大会环节
1. **大会开幕**: 狮王致辞欢迎
2. **市场回顾**: 年度股价走势、涨跌幅
3. **财富排行**: 所有动物的盈亏排名
4. **慈善捐款**: 盈利动物捐款环节
5. **代表发言**: 盈利/亏损代表发言
6. **精彩节目**: 动物们的才艺表演
7. **大会闭幕**: 狮王闭幕致辞

#### 慈善捐款规则
- 捐款条件: 年度盈利的动物
- 捐款金额: 利润的10%（最低¥100，最高¥5,000）
- 捐款奖励: 获得4个月的公告头条播报特权

#### 历史记录
- 点击顶部"🎪 年度大会"按钮可查看历史记录
- 支持查看任意年份的年度大会内容

## 💡 启动经验总结

### 常见问题与解决方案

#### 1. 端口被占用
如果默认端口被占用，可以指定其他端口：
```bash
npm run dev -- --port 3000
```

#### 2. 依赖安装失败
尝试清除缓存后重新安装：
```bash
# 清除npm缓存
npm cache clean --force

# 删除node_modules和lock文件
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

#### 3. Windows系统启动命令
Windows系统推荐使用以下命令启动：
```bash
# 方式1：直接启动
npm run dev

# 方式2：指定端口
npm run dev -- --port 8080

cd e:\ai_learn\animalgame\nocode ; 

npm run dev

# 方式3：使用yarn（如已安装）
yarn dev
```

#### 4. 浏览器访问
启动成功后，在浏览器中访问：
- 默认地址：http://localhost:5173
- 指定端口：http://localhost:8080

#### 5. 热更新不生效
如果热更新不生效，尝试：
```bash
# 重启开发服务器
# 按 Ctrl+C 停止，然后重新运行
npm run dev
```

#### 6. 构建报错
如果构建时报错，检查：
- Node.js版本是否符合要求（建议使用LTS版本）
- 依赖是否完整安装
- 磁盘空间是否充足

### 推荐开发环境
- **IDE**: VS Code
- **Node.js**: v18.x LTS 或更高版本
- **包管理器**: npm 或 yarn
- **浏览器**: Chrome、Firefox、Edge（现代浏览器）

### VS Code 推荐插件
